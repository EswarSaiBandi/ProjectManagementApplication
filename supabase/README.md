# Supabase Configuration

This project uses Supabase for the backend database and authentication.

## Setup

1.  We use the Supabase CLI for local development and migrations.
2.  Run `npx supabase login` to log in.
3.  Run `npx supabase init` if not already initialized (though this folder should contain the config).

## Environment Variables

See `.env.example` in the respective apps for required keys.
